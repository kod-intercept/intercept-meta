# Contributing to intercept-meta

Welcome, and thank you for your interest in the INTERCEPT project!

This repository (`intercept-meta`) is maintained by **Končar Digital (KOD)**
and serves as the central metadata and documentation hub for the INTERCEPT
platform. It is **read-only** for all external collaborators.

## Contribution Policy

- Direct code or documentation changes (commits/pushes) are **not accepted**
  from external users.
- If you have suggestions, feedback, or corrections:
  - Open a GitHub **Issue** with a clear and concise description.
  - Use labels if applicable (e.g., `suggestion`, `documentation`, `question`).

Optionally, you may:
- Fork the repository.
- Propose changes via a **Pull Request**.
  - Note: All pull requests will be reviewed and considered at the discretion
    of the KOD maintainers.

## Scope of this repository

This repository contains:
- Organizational and architectural documentation.
- Project-level structure and contact information.
- References to relevant platform components.

It **does not** contain:
- Executable code.
- Project-specific configurations.
- Development or build infrastructure.
