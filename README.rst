KOD INTERCEPT GitHub Organization
=================================

This GitHub organization hosts repositories related to the INTERCEPT project,
developed and maintained by Končar Digital (KOD).

Overview
--------

The ``kod-intercept`` organization is responsible for the core software
components, integration adapters, and technical documentation of the INTERCEPT
threat sharing platform (TSP) and Končar Aggregation Platform (AP). Only the
KOD team maintains write access to the repositories, while selected partners
may have read access to specific parts of the codebase.

The following repositories are currently maintained within this organization:

- ``intercept-meta`` *(public)*

  Contains the project-wide documentation, collaboration guidelines, and
  organizational structure. This repository acts as the entry point for all
  stakeholders in the INTERCEPT ecosystem.

- ``intercept-tsp-platform`` *(private/shared)*

  Core implementation of the Threat Sharing Platform (TSP), which serves as the
  central integration point between partners' Local MISP instance and other
  partner-specific components. This platform is designed to enhance
  collaboration and streamline data sharing across various partner networks.

- ``intercept-tasks`` *(private)*

  Internal task management and coordination repository for the KOD team. It
  tracks development progress and technical planning across all INTERCEPT 
  components.

More repositories may be added in the future to cover additional adapters,
client interfaces, testing infrastructure, and partner-specific integrations.

Access Policy
-------------

- Only the KOD team can push to repositories.
- Partners may be granted **read-only access** to selected shared repositories.
- Internal or infrastructure-related repositories remain **private**.
